# What is about-me?

About Me,a web application By Spring Boot 2.3.0

# How to use this image

## Docker Pull Command

```bash
docker pull fengwenyi/about-me:1.0.0
```

## start a redis instance

```bash
docker run -d -p 8080:8080 --name about-me -d fengwenyi/about-me:1.0.0
```
