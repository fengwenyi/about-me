# About Me

## 概述

一个基于Spring Boot 的 WEB 应用!

## 快速开始

### 拉取

```bash
docker pull fengwenyi/about-me:1.0.0
```

### 运行

```bash
docker run -d -p 8080:8080 --name about-me -d fengwenyi/about-me:1.0.0
```
